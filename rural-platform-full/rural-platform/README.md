# Rural Platform Web App

## Tech Stack
- Frontend: React.js, Material UI, Axios
- Backend: Node.js, Express, MongoDB

## Features
- Responsive homepage
- User signup/login (JWT)
- User dashboard (bookings, profile update)
- Service and product listings
- News/updates display
- Booking function

## Setup (Local)
1. `cd backend && npm install && npm start` (edit `.env`)
2. `cd frontend && npm install && npm start`
